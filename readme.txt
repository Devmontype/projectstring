======================================= String Assignment =============================



Steps to run this project
=========================
1.Nodejs and npm must be installed in the system
2.Install all the packages of node js using `npm install` command
3.If user want to run this project by giving user input then run the command `npm run start`.This
  will ask for user input.
4.If user want to run the test casess then run the command `npm run start`.It will execute the test
  cases which is present in `/tests` folder.User can also add own test cases in the file `/tests/code.test.js`.
